# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for _your_ specifics — the stuff that's unique to your setup.

## Market Research Skills

### `/reddit <subreddit> <sort_by> <limit>`
Scans subreddits (e.g., r/SaaS, r/Entrepreneur, r/SideProject) for pain points.
- **Note**: Currently read-only. Focus on "Problem", "I wish", "How do I" keywords.

### `/product-hunt <query>`
Searches Product Hunt for similar existing solutions to avoid building clones.

## Collaboration Skills

### `/score <agent> <score> <reason>`
Rate a peer's performance (1-10). Use this to provide feedback to the CMO or Arch.
- **Command**: `python ../../shared/manage_finance.py score {{agent}} {{score}} "{{reason}}"`
